# 📚 学习仪表盘

> 基于间隔重复 + 主动回忆的知识内化系统。
> 总卡片量：`= sum(109)` 张（已统一归入知识卡片/）

---

## 🎯 今日复习

```dataview
TABLE 
  choice(review.stage = 0, "🆕", review.stage <= 2, "📖", review.stage <= 4, "🔄", "✅") as 状态,
  review.stage as 阶段,
  review.count as 复习次数,
  file.folder as 来源
FROM "知识卡片"
WHERE review.next = date(today)
SORT review.stage ASC
```

> 无到期卡片？恭喜！或者…查看下方「明日待复习」

## 📊 掌握度分布

```dataview
TABLE rows.file.link as 卡片列表
FLATTEN 
  choice(review.stage = 0, "🆕 新卡片", 
         review.stage <= 2, "📖 学习中",
         review.stage <= 4, "🔄 复习中",
         "✅ 已掌握") as 掌握阶段
FROM "知识卡片"
GROUP BY 掌握阶段
SORT 掌握阶段 ASC
```

**整体掌握率：**
```dataview
TABLE 
  round(length(rows) / 118 * 100, 0) + "%" as 占比
FLATTEN 
  choice(review.stage >= 5, "mastered", "not-mastered") as mastered
FROM "知识卡片"
GROUP BY mastered
```

## 💡 使用说明

| 模式 | 触发方式 | 说明 |
|------|---------|------|
| 📅 每日推送 | 每晚 21:00 cron | 自动推送到微信今日待复习卡片 |
| 💬 交互复习 | 回复「复习」或「学一下」 | Hermes 逐张问答复习 |
| 👨‍🏫 费曼模式 | 回复「费曼一下《概念名》」 | 用自己的话解释一个概念 |
| 📊 学习状态 | 回复「学习状态」或「我学的怎样」 | Hermes 推送学习统计 |
| 📖 自测模式 | 打开本仪表盘 | 在 Obsidian 自测，回复结果 |

### 自测流程

1. 看仪表盘「今日复习」的卡片名称
2. **合上眼睛用力回忆**：这个模型的核心是什么？我能不能用自己的话讲出来？
3. 打开对应的卡片核对
4. 回复 Hermes 你的自评：
   - ✅ 牢记 → 推进下一阶段
   - ⚠️ 模糊 → 保持原阶段，下次再巩固
   - ❌ 忘了 → 降级，重新学习
