## 📋 学习记录内容要求

### 必填部分
1. **学习日期**：YYYY-MM-DD格式
2. **学习主题**：sXX - 课程名称
3. **学习目标**：3-5个具体目标
4. **代码实践**：运行记录和结果
5. **学习心得**：遇到的问题和收获
6. **进度评估**：知识掌握程度自评

### 建议部分
1. **实践练习**：完成的代码修改
2. **相关链接**：关联的笔记和资源
3. **总结反思**：今日完成情况和改进
4. **明日计划**：下一课程的学习安排

## 🆘 常见问题

### Q1：脚本无法执行？
```bash
# 检查脚本权限
chmod +x 创建学习记录.sh

# 检查脚本路径
cd "/Users/jansen/Documents/obsidian/JansenObsidian/4.学习资料/Agent工程师/1.学习资源/Claude Code学习"
```

### Q2：文件已存在？
脚本会提示是否覆盖，可以选择：
- `y`：覆盖现有文件
- `n`：取消操作，手动处理

### Q3：课程编号错误？
课程编号格式应为：`s01`, `s02`, ..., `s12`
```bash
# 正确
./创建学习记录.sh s01 "Agent循环机制"

# 错误
./创建学习记录.sh 1 "Agent循环机制"
```

### Q4：如何修改模板？
1. 编辑[[学习记录模板]]文件
2. 修改后，新创建的文件会使用新模板
3. 已有文件需要手动更新
