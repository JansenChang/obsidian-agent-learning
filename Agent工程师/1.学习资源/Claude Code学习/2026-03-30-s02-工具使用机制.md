## 📝 学习笔记

### 核心概念理解

#### 1. 什么是工具使用机制？
- **定义**：Agent调用外部工具来完成特定任务的能力
- **类比**：类似于人类使用计算器、搜索引擎等工具
- **价值**：扩展Agent的能力边界，完成更复杂的任务

#### 2. 工具调用的基本流程
```
用户请求 → Agent分析 → 工具选择 → 参数准备 → 
工具调用 → 结果获取 → 结果处理 → 用户响应
```

#### 3. 工具的类型
- **计算工具**：数学计算、单位转换等
- **查询工具**：数据库查询、API调用等
- **转换工具**：格式转换、数据清洗等
- **控制工具**：系统控制、设备操作等

#### 4. 工具注册机制
- **静态注册**：在代码中直接定义工具
- **动态注册**：运行时动态添加工具
- **工具发现**：自动发现可用工具
- **权限控制**：工具访问权限管理

### 代码分析

#### 关键代码片段
```python
# 示例：工具注册
def register_tools(agent):
    """向Agent注册工具"""
    
    # 注册计算器工具
    @agent.tool(name="calculator", description="执行数学计算")
    def calculator(expression: str) -> str:
        """计算数学表达式"""
        try:
            result = eval(expression)
            return f"计算结果: {result}"
        except Exception as e:
            return f"计算错误: {e}"
    
    # 注册天气查询工具
    @agent.tool(name="weather", description="查询天气信息")
    def weather(city: str) -> str:
        """查询指定城市的天气"""
        # 调用天气API
        return f"{city}的天气信息..."
    
    return agent
```

```python
# 创建一个时间查询工具
@agent.tool(name="current_time", description="获取当前时间")
def get_current_time(timezone: str = "Asia/Shanghai") -> str:
    """获取指定时区的当前时间"""
    from datetime import datetime
    import pytz
    
    try:
        tz = pytz.timezone(timezone)
        current_time = datetime.now(tz)
        return f"{timezone}的当前时间是: {current_time.strftime('%Y-%m-%d %H:%M:%S')}"
    except Exception as e:
        return f"时区错误: {e}"
```

### 练习3：工具调用日志
```python
# 添加工具调用日志记录
import json
from datetime import datetime

def log_tool_call(tool_name: str, args: dict, result: str, success: bool = True):
    """记录工具调用日志"""
    
    log_entry = {
        "timestamp": datetime.now().isoformat(),
        "tool": tool_name,
        "arguments": args,
        "result": result,
        "success": success
    }
    
    # 保存到日志文件
    with open("tool_calls.log", "a") as f:
        f.write(json.dumps(log_entry, ensure_ascii=False) + "\n")
    
    return log_entry
```
