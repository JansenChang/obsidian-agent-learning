## 📝 学习笔记

### 核心概念理解

#### 1. Agent循环是什么？
- **定义**：Agent与外部环境（Harness）之间的持续交互过程
- **类比**：类似于人类对话的"听-思考-说"循环
- **关键**：保持状态，持续响应

#### 2. 消息传递机制
- **输入消息**：用户或系统的请求
- **处理过程**：Agent内部逻辑处理
- **输出消息**：Agent的响应
- **状态保持**：对话上下文的管理

#### 3. Harness的作用
- **接口层**：连接Agent与外部世界
- **消息路由**：转发输入和输出
- **状态管理**：维护会话状态
- **错误处理**：处理异常情况

### 代码分析

#### 关键代码片段
```python
# 示例：基本的Agent循环
while True:
    # 1. 获取输入
    user_input = get_input()
    
    # 2. 检查退出条件
    if user_input.lower() == "quit":
        break
    
    # 3. 处理输入
    response = agent.process(user_input)
    
    # 4. 输出响应
    print(response)
```

#### 设计模式
- **观察者模式**：Harness监听Agent状态变化
- **状态模式**：管理对话的不同阶段
- **策略模式**：不同的消息处理策略
