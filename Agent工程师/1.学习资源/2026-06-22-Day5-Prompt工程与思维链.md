# Day5 — Prompt工程与思维链

**日期：** 2026-06-22
**前置：** Day1 Attention → Day2 Decoding → Day3 Tokenization → Day4 KV Cache

## 核心概念

- **CoT (Chain-of-Thought, 思维链)** — 2022年Wei等人提出，在输出答案前先输出推理步骤
- **Zero-shot CoT** — 加一句「请一步步思考」触发推理，涌现能力
- **Few-shot CoT** — 给1-3个完整示例（问题+推理+答案），引导模型模仿
- **Self-Consistency** — 多次采样投票，提升准确率（GSM8K 70%→90%+）
- **ReAct** — CoT + 工具调用，Thought→Action→Observation 循环
- **ToT (Tree-of-Thought, 思维树)** — 多分支推理+回溯搜索，适合24点/创意写作

## 知识关联

- Attention 提供上下文理解 → Decoding 决定选词 → KV Cache 加速生成 → **CoT 引导推理路径**
- CoT 是 ReAct 的子组件（推理部分）
- ToT 是 CoT 的升级版（链→树）
