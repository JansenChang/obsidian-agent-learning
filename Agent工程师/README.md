# Agent工程师学习系统

## 📚 系统设计

这是一个**循序渐进 + 思维导图构建**的学习系统，采用**我创建内容 → 你学习记录 → 自查巩固 → 导图整合**的四步法：

### 1. 学习资源 (`1.学习资源/`)
- **我创建**：每天为你创建当天的学习内容
- **包含**：学习主题 + 思维导图
- **文件格式**：
  - `YYYY-MM-DD-学习主题.md` - 详细学习内容
  - `YYYY-MM-DD-思维导图.md` - 当日知识思维导图

### 2. 学习日记 (`2.学习日记/`)
- **你记录**：记录每天的学习过程、心得、问题
- **模板化**：使用模板快速记录
- **文件格式**：`YYYY-MM-DD-学习日记.md`

### 3. 题库自查 (`3.题库自查/`)
- **你答题**：根据学习内容回答对应日期的问题
- **我检查**：告诉我你的答案，我帮你查漏补缺
- **文件格式**：`YYYY-MM-DD-自查题目.md`

### 4. 思维导图系统
- **总体导图**：`思维导图-Agent工程师知识体系.md` - 完整知识体系
- **每日导图**：`1.学习资源/YYYY-MM-DD-思维导图.md` - 当日知识导图
- **渐进构建**：每天学习后，将当日导图整合到总体导图中

## 🔄 每日学习流程（含思维导图）

1. **早上**：我创建当天的学习资源（含思维导图）→ 你开始学习
2. **学习中**：你在学习日记中记录过程
3. **导图学习**：通过思维导图建立知识体系
4. **学习后**：你回答题库中的问题
5. **导图整合**：将当日思维导图内容整合到总体思维导图中
6. **检查**：告诉我你的答案，我帮你完善
